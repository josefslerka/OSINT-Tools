## Nástroje pro analýzy webů
- Ahrefs - analýza zpětných odkazů https://ahrefs.com/
- SimilarWeb - odhady návštěvnosti a podrobnosti o publiku https://similarweb.com/
- AnalyzeID - s kým stránka sdílí GA a další kódy https://analyzeid.com/
- BuzzSumo - jaké články trendovaly na sociálních sítítch https://buzzsumo.com/
- Feedly: RSS čtečka s API, kterou jde napojit do Zapieru a udělat tak automatizovaný monitoring https://feedly.**com**
